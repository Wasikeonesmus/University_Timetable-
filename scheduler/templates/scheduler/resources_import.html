💡 Import Schema Templates

Files must have headers in the first row. The system automatically cleans extra spaces, capitalization, and resolves structural objects.

Rooms Schema

Header Type Required
nameTextYes
capacityNumberYes
room_typeLecture / LabYes

Lecturers Schema

Header Type Required

⚡ Smart Import (Auto-Detect)

Upload any Excel (.xlsx) or CSV (.csv) file. The system will automatically detect the structure:

  • Flat Timetable: A list of sessions containing Day, Time, Room, Course, and Lecturer.
  • Separate Entity Sheets: Multiple sheets named Rooms, Lecturers, Courses, etc.
  • Single Sheet list: A list of courses, rooms, or lecturers.
How it works: Columns are matched using 200+ aliases (e.g. "VENUE" mapped to "Room"). You will review the detected entities on a preview screen before anything is written to the database.

⚡ Import All at Once — Single Excel File

Upload one Excel (.xlsx) file with up to 4 sheets. The system automatically detects each sheet by its name and imports everything in the correct order.

Sheet Name Also Accepted As Required Columns
Rooms Venues, Locations, Halls venue/name, capacity/seats, type
Lecturers Staff, Teachers, Instructors name/instructor, email
Students Groups, Classes, Cohorts, Sections student/name, size/enrolment
Courses Units, Modules, Subjects code, name, hours/duration, type
⚠ Note: Only Excel (.xlsx) files are supported for multi-sheet import. Column headers are flexible — use any common name (e.g. venue instead of room, enrolment instead of size).
